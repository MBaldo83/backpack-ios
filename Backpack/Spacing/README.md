# Backpack/Spacing

## Usage

### Objective-C

```objective-c
#import <Backpack/Spacing.h>

CGFloat myAppSpacingSmall = BPKSpacingSm;
CGFloat myAppSpacingLarge = BPKSpacingLg;
```

### Swift

```swift
import Backpack

let myAppSpacingSmall = Backpack.BPKSpacingSm
let myAppSpacingLarge = Backpack.BPKSpacingLg
```
