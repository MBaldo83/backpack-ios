# Backpack/BorderSize

## Usage

### Objective-C

```objective-c
#import <Backpack/BorderSize>

CGFloat myAppBorderBorderSmall = BPKBorderWidthSm;
CGFloat myAppBorderBorderLarge = BPKBorderWidthLg;
```

### Swift

```swift
import Backpack

let smallBorder = BPKBorderWidthSm
let largeBorder = BPKBorderWidthLg
```
